# AI Health & Fitness Coach Dashboard

A premium, interactive fitness dashboard with AI-powered coaching, workout tracking, nutrition planning, and progress monitoring.

## Features

- **AI Fitness Coach**: Personalized workout and nutrition guidance
- **Authentication System**: Secure login/signup functionality
- **Workout Tracking**: Comprehensive exercise routines and tracking
- **Diet Plans**: Personalized nutrition plans and recipes
- **Progress Monitoring**: Detailed metrics and achievement tracking
- **Responsive Design**: Works on all device sizes
- **Premium UI**: Glassmorphism design with neon accents
- **Interactive Elements**: Charts, progress bars, and data visualization

## Technologies Used

- HTML5
- CSS3 (with modern features like backdrop-filter)
- JavaScript (ES6+)
- Chart.js for data visualization
- Font Awesome for icons
- Responsive design principles

## Installation & Usage

1. Clone the repository:
```bash
git clone https://github.com/Moizali011/Health-Agent.git
```

2. Navigate to the project directory:
```bash
cd Health-Agent
```

3. Run the local server:
```bash
node server.js
```

4. Open your browser and go to:
```
http://localhost:3009
```
